# NovelOS Studio

NovelOS Studio is a modern, web-based creative writing companion designed to help you focus on what matters most: writing your story. It provides a clean, distraction-free interface with powerful tools to streamline your writing process from the first draft to the final manuscript.

![NovelOS Studio Screenshot](https://storage.googleapis.com/aifirebase/screenshots/novelos-screenshot-2.png)

## Features

- **Rich Text Editor:** A WYSIWYG editor that supports common formatting options like bold, italics, and underline.
- **AI-Powered Proofreading:** An integrated AI assistant to check your grammar and style, offering corrections to improve your writing.
- **Chapter Management:** Easily add, delete, rename, and reorder chapters with a simple interface in the sidebar.
- **World-Building & Notes:** A dedicated "Notes" tab with organized sections for characters, locations, and plot points to keep your ideas structured.
- **AI Prompt Generator:** Spark your creativity with an AI-powered prompt generator that creates unique writing prompts based on your chosen genre, theme, and keywords.
- **Export to PDF & DOCX:** Export your entire manuscript to universally compatible formats.
- **Font Selection:** Customize your writing experience by choosing from a selection of elegant fonts.
- **Light & Dark Modes:** Switch between light and dark themes for a comfortable writing environment, day or night.
- **Local Storage:** All your work is automatically saved to your browser's local storage, so you'll never lose your progress.
- **Cover Creator:** A mock-up tool to visualize your book cover.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18 or later)
- npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/NovelOS-Studio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
   npm run dev
   ```
4. Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [ShadCN UI](https://ui.shadcn.com/)
- **AI Integration:** [Genkit](https://firebase.google.com/docs/genkit)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
